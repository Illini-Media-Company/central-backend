{# This file defines HTML that can be imported into any template by adding: include "partials/loading.html" inside of curly braces with % on either side. Defines a loading screen with customizable text. To call this script in another file: showLoading(text) Where text is a string. It should be only a few words. This function does not return a value. This function can be called repeatedly with different text values to change the text on the existing loading screen. Calling hideLoading() will remove the loading screen.
