{# This file defines a macro for adding paginated scroll for pages rendered server-side. To access it in a file, add the following outside of any other Jinja block: import "macros/pagination.html" inside of curly braces with % on either side. There is no CSS file for this macro. To use it in-place in a file, put the following inside double curly braces: pagination(xxx, yyy, zzz, "www") Where xxx should be replaced with the page that is currently loaded. yyy should be replaced with the total number of items, zzz should be replaced with the number of items to display per page. www should be replaced with the URL parameter that the view function will check for. All of the above must be Jinja variables passed into the template from the view function. For an example, look at templates/employee_agreement_admin.html Created by Jacob Slabosz on Nov. 16, 2025 Last modified Nov. 16, 2025 #} {% macro pagination(page, total, per_page, param) %}
{% if page > 1 %} Prev. {% endif %}

Page {{ page }} of {{ (total + per_page - 1) // per_page }}

{% if page * per_page < total %} Next {% endif %}
{% endmacro %}
